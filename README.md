# libwrapper-types

TypeScript declaration-only package for the `libWrapper` Foundry module API.

## Install (GitHub)

```sh
npm install --save-dev github:henry-malinowski/libwrapper-types#main
```

## Usage

Add the package to your `tsconfig.json` `types` array:

```json
{
  "compilerOptions": {
    "types": ["fvtt-types/lenient", "libwrapper-types"]
  }
}
```
